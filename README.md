# English on the Road Pro

Plataforma web estática para práctica de inglés de operadores B1.

## Uso
1. Abre `index.html`.
2. Para GitHub Pages, sube toda la carpeta al repositorio.
3. Sustituye los audios de demostración dentro de `assets/audio/`.

## Funciones
- Panel del operador
- Ruta de aprendizaje
- Lección funcional DOT
- Comprensión auditiva
- Pronunciación por fragmentos
- Reconocimiento de voz Web Speech API
- Simulación
- Progreso con localStorage
- Panel docente
- Calificaciones
- Observaciones
- Exportación CSV

## Audio
La ruta esperada para la primera demostración es:
`assets/audio/dot-inspection/turn-off-your-truck.mp3`

Mientras el archivo no exista, la plataforma usa voz del navegador solo como respaldo de demostración.


## Cambio autodidacta
La primera misión ahora enseña explícitamente cada respuesta antes de pedirla:
- escucha del prompt,
- significado,
- explicación de uso,
- práctica por fragmentos,
- práctica completa,
- simulación final.

Ninguna simulación exige una respuesta que no haya sido presentada y practicada previamente dentro de la plataforma.
